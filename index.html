Box Model

1. Que es Box model

el modelo de caja se forma en los navegadores apartir de cualquier elemento que se encuentre en nuestro html este se vera en forma restangular, el modelo de caja tiene 4 partes:

  • Content
  • Padding
  • Border
  • Margin

    • 2.  Que es padding , con ejemplo (4 lados mas shortcuts)

    el padding es el area alrededor del contenidoy el borde de la caja, se tiene:

  • padding-top: 10px;
  • padding-right: 25px;
  • padding-bottom: 20px;
  • padding-left: 5px;

    • caja
    en este caso el color azul.

    3.  Que es margin , con ejemplo (4 lados mas shortcuts)

    Es la separación entre una caja y otras cajas adyacentes. ejemplo de como aplicarlos:

  • margin-top: 20px;
  • margin-right: 20px;
  • margin-bottom: 20px;
  • margin-left: 15px;
    • los valores tambien pueden ser negativos.

    4. Que es border, y aplicarlo 

    el border es el que rodea la caja se peude aplicar:

  • border: solid 5px red.

    • 5. Para que es el uso del selector universal en el box model.

    el selector universal en el box model se utiliza para poder utilizar el:

  • box-sizing: content-box;
    • el cual permite incluir el contenido, pero no incluyen el relleno,borde o el margen.
